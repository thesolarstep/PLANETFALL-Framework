# Contributing to PLANETFALL

PLANETFALL is a scientific framework built on the principle of adversarial validation. We welcome contributions that improve the rigor and skepticism of the exoplanet validation process.

## Our Philosophy
"We built this because Source 382 lied to us." 

In PLANETFALL, every candidate enters as guilty. Your contributions should help the framework more effectively "prosecute" candidates to ensure only the most robust signals survive.

## How to Contribute
1. **Focus on Falsification**: We prioritize modules that can identify new types of false positives (EBs, background blends, instrumental artifacts).
2. **Code Standards**: 
   - Ensure all modules follow the established adversarial patterns.
   - Maintain type safety and thorough documentation.
   - Do not include any proprietary or private data in your PRs.
3. **Scientific Integrity**: All new methods should be grounded in established astronomical literature or rigorous statistical foundations.

## Reporting Bugs
If you find a loophole where a known false positive might pass, please open an issue immediately. In PLANETFALL, a false negative (missing a planet) is always preferable to a false positive (claiming a fake planet).
